OpenPNE2 for Smart Phoneは、オープンソースのSNSエンジン「OpenPNE2」を、スマートフォン向けに対応させたものです。

【このような方にお勧め】
・ＳＮＳを携帯で使っていたメンバーがスマートフォンに買い換えて、
　ログインできなくなった
　写真投稿ができなくなった
　画面が見ずらくなった
……など、不便を感じていたＳＮＳオーナーの方。

・スマートフォンに対応するため、OpenPNE3に乗り換えようとしたがうまくいかなかったり、サイトが重くなってしまったので、断念したＳＮＳオーナーの方。

・これから運用を開始するＳＮＳを、スマートフォンから使えるようにしたいオーナーの方にも。

動作が軽く、セットアップが簡単なＳＮＳ構築ソフトウェアとして実績のあるOpenPNE2に、スマートフォン向けに対策をした「OpenPNE2 for Smart Phone」。
ぜひご利用ください。


【主な特徴】
１．携帯電話のメールアドレスドメイン、ＰＣメールアドレスドメインの両方で登録/ログイン可能です。
従来のOpenPNE2ではできなかった、携帯メールアドレス＋スマートフォンのみでのＳＮＳへの登録・利用が可能です。

２．写真のメール投稿に対応しました。
iPhone/iPadは、デフォルトブラウザからの写真アップロードができませんが、メールにより写真投稿を可能にしました。
メール投稿のアドレスも、携帯メール/ＰＣメールの両方に対応しています。

３．「もっと読む」の動的表示
日記やコメントの「もっと読む」では、画面全体を更新するのではなく、続きの記事だけを読み込んで表示します。

４．jQuery Mobileを用いた、スマートフォン向けのユーザインターフェース（ＵＩ）
スマートフォン画面は、jQuery Mobileで作成しました。
jQuery Mobuleは、スマートフォンの画面サイズ・タッチ操作に適したＵＩを提供する、開発フレームワークです。
iPhone、Androidなど、多種多様なデバイスで、サイトを同様のＵＩで表示できます。


【対応機能】
スマートフォン表示に対応した機能は、以下の通りです。

ＳＮＳ登録、日記、コミュニティ、トピック／イベント、メッセージ、マイフレンド機能、招待機能、プロフィール、設定変更

対応していない機能の画面を表示しようとした場合は、既存のＰＣ画面レイアウトで表示されます。


【セットアップ方法】
OpenPNE2のセットアップガイドは、setup/以下を参照してください。

１．新規インストールの場合
OpenPNE2のセットアップ方法に準じます。
OpenPNE2のインストールガイドと同様に行ってください。

２．運用中のＳＮＳをアップグレードする場合
カスタマイズしていないＳＮＳであれば、OpenPNE2のアップグレードに準じます。
アップグレードSQLの実行は必要ありません。
ソースコードを設置して、config.phpを設定してください。

ソースをカスタマイズしたＳＮＳについては、動作を保証しません。
